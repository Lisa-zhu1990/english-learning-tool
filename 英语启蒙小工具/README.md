# 英语启蒙小工具（离线版）

这是一个给家长陪孩子学习英语使用的离线 HTML 小工具。

## 使用方式

打开：

`current/英语听写工具.html`

双击即可运行，不需要联网、不需要安装、不依赖服务器。

## 当前版本

当前版本：`V2.3.3`

主要能力：

- 教材词库选择
- 自定义词库
- 英语听写
- 家长人工批改
- 本轮纠错
- 薄弱词库
- 学习档案
- 自然拼读展示
