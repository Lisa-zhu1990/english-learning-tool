# CHANGELOG V2.1.2

## 新增

- 新增顶部“词库来源”：教材 / 自定义。
- 新增 `currentSource`，明确记录当前来源：教材来源包含 book、unit、range；自定义来源只读取用户输入。
- 新增 `currentPracticeWords` 当前练习词库，学习展示、听写、人工批改、错词复练共用同一份数据。
- 新增家长人工批改：听写后勾选写错的词，生成 `wrongWords` 用于错词复练。

## 修改

- “生成学习”按当前来源生成练习词库：教材模式读取 `LIBRARY`，自定义模式读取 textarea。
- 教材模式不再把 Unit / 分组 / 随机结果写入自定义词库 textarea。
- 当前词库展示改为使用原“本轮结果”区域，未批改前显示实际练习词列表。
- 随机 10 词和全部随机会展示实际生成结果。
- 听写流程继续只播放完整单词，并使用 `currentPracticeWords`。

## 修复

- 修复教材模式和自定义词库模式混用的问题。
- 修复教材筛选结果污染自定义 textarea 的问题。
- 修复生词展示、听写、错词复练数据来源不一致的问题。
- 修复错词复练后批改勾选列表未重建的问题。

## 移除

- 移除拍照上传入口。
- 移除 OCR/拍照批改相关界面文案，改为家长人工批改。
